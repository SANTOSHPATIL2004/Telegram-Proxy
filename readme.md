## 📋 Using the Proxy List

To use the proxies collected with your Telegram app:

1. Open your Telegram application on Windows.
2. Go to **Settings** > **Advanced** > **Connection Type**.
3. Choose **Use Proxy**.
4. Enter one of the proxy addresses saved in `proxies.txt`.
5. Save and connect.

You can try different proxies from the list if one does not work.
